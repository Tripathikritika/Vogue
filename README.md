## Vogue - Womens Website

The project is all about Womens Beauty product. It includes products for skin care , hair care and nail care.

## Motivation

The private routing sprint project @Masai School.

## Features

- [x] React Router
- [x] Private Routing
- [x] Adding into cart
- [x] Cart length visibile
- [x] Mock Server used
